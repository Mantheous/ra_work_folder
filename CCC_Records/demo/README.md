# Sample of Civilian Conservation Corps Records

This is the information for Ladislaus Giza, a member of the Civilian Conservation Corps.

This is an example of the kind of data we could get from the Civilian Conservation Corps records. More information on these records can be found [here](https://catalog.archives.gov/id/12034477). These records contain a lot of different types of information.

The things that I think are particularly interesting are the medical information and the court proceedings. 

## Data Extraction Process
This data was extracted by copying the extracted text from the National Archives webpage and then having Gemini create a json file from the text. The images were downloaded programatically with the API.

## Format
- [Json file](Ladislaus_Giza.json) 
- [Extracted text](Ladislaus_Giza_extracted_text.txt) 
- Images are in [source_images](source_images)
