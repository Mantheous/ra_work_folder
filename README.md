# ra_work_folder
Scrapers, scripts and fun toys for my work in the RLL

## Project Structure
- `Arkaie_Scraper/`: Contains the base scraper class and common tools for websites that use the Arkaïe platform. These scripts serve as the foundation for the specific department scrapers.
- `Civil_Status/`: Contains the civil status web scrapers organized by department (e.g., Aube1, Aube2, Creuse, etc.). Each directory has specific scraper configurations.
- `Utilities/`: Contains utility scripts for downloading, validating CSVs, browser actions, and notifications.
- Old and unused code has been marked with a comment `# OLD AND UNUSED CODE` at the top of the files.
